# 多Agent协同运营自动化系统

## 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                        Manager Agent                            │
│                    (任务协调与流程控制)                           │
└─────────────────────────────────────────────────────────────────┘
                    │                │                │
                    ▼                ▼                ▼
         ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
         │   Researcher │  │    Writer    │  │    Editor    │
         │   Agent      │  │    Agent     │  │    Agent     │
         │  (热点分析)   │  │  (内容创作)   │  │  (润色排版)   │
         └──────────────┘  └──────────────┘  └──────────────┘
                                                         │
                                                         ▼
                                              ┌──────────────┐
                                              │  Publisher   │
                                              │    Agent     │
                                              │   (多平台发布) │
                                              └──────────────┘
```

## Agent 职责

| Agent | 角色 | 核心能力 |
|-------|------|----------|
| Manager | 协调者 | 任务分配、进度跟踪、异常处理 |
| Researcher | 研究员 | 热点挖掘、竞品分析、选题推荐 |
| Writer | 写作师 | 文章撰写、标题生成、风格适配 |
| Editor | 编辑师 | 文字润色、排版优化、图片建议 |
| Publisher | 发布师 | 多平台发布、定时任务、效果追踪 |

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行系统
python main.py

# 或者自定义运行
python main.py --topic "AI Agent发展趋势" --platforms "微信公众号,知乎"
```

## 配置说明

编辑 `config.yaml` 自定义：
- API 密钥
- 发布平台账号
- Agent 参数
- 定时任务设置
