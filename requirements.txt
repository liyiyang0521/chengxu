# 多Agent协同运营自动化系统

## 依赖

```
aiofiles>=0.8.0
pyyaml>=6.0
requests>=2.28.0
openai>=1.0.0
anthropic>=0.5.0
python-dateutil>=2.8.0
```

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
# 交互模式
python main.py

# 演示模式
python main.py -m demo

# 指定主题
python main.py -t "AI Agent发展趋势分析"

# 详细输出
python main.py -v
```
